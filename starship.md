# Starship
https://starship.rs/

## Installation

### Windows

- winget: `winget install --id Starship.Starship`

### Linux

- all: `curl -sS https://starship.rs/install.sh | sh`
- arch: `pacman -S starship`
- pacstall: `pacstall -I starship-bin`
